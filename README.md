# NJU_Health-Checkin

**使用方法**

1. 在config.json中填写南大统一身份认证的学号和密码
2. 确保依赖库安装完整，`pip install -r requirements.txt`
3. `python checkin.py`后台运行即可

运行后立即完成一次打卡，此后每天00:01分自动打卡

:rotating_light:**请务必如实上报健康状况**，如有异地出行、身体状况变动、本人或家人健康码非绿色，请停止使用此脚本。
